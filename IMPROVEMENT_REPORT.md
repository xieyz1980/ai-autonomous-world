## 🚀 部署建议

### 云主机配置建议

**最低配置**:
- CPU: 2核
- 内存: 4GB
- 存储: 50GB SSD
- 带宽: 5Mbps

**推荐配置**:
- CPU: 4核
- 内存: 8GB
- 存储: 100GB SSD
- 带宽: 10Mbps

### 部署步骤（简化版）

```bash
# 1. 准备云主机并安装Docker

# 2. 克隆代码
git clone https://github.com/xieyz1980/ai-autonomous-world.git
cd ai-autonomous-world

# 3. 配置环境变量
cp .env.example .env
vim .env  # 填入真实的OPENAI_API_KEY

# 4. 一键部署
./deploy.sh

# 5. 验证
./test.sh
```

### 监控建议

**长期运行监控**:
- 使用monitor.py持续监控
- 定期检查monitor-reports.jsonl
- 设置告警阈值（如健康检查失败>3次）

**关键指标关注**:
- 节点在线率
